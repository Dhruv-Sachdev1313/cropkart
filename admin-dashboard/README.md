# Welcome to CropKart!

> CropKart is a clean, modern and intuitive PWA for customers to connect with farmers and buy fresh produce directly from them

### Technology Stack

| Usage | Frontend |
| :---: | :---: |
| Framework | Next.js (React) |
| Language | TypeScript |
| State Management | Redux |
| Styling | Tailwind CSS |
| Hosting | Vercel |
